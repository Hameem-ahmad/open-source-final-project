UMT Student Management

About the System

The Student Management System is a full-stack web application developed as the Final Semester Project for the Open Source Software Development course at the University of Management and Technology.

It addresses the challenges of paper-based and spreadsheet-driven student administration by providing a centralized platform for managing student records, enrollments, attendance, grades, and announcements.

Technology Stack

Frontend

  • HTML, CSS, JavaScript
  • Deployed on Vercel / Netlify

Backend

  • FastAPI (Python)
  • SQLAlchemy ORM
  • JWT Authentication
  • Deployed on Render

Database

  • PostgreSQL
  • Hosted on Supabase

Version Control

  • Git & GitHub
